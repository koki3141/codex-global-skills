# Visual Experiment Design Prompt

研究対象と既存資料を確認し、学術的価値、反証可能性、実装可能性が最も高い研究テーマを1つ決定してください。中心RQ、仮説、因果構造、独立・従属・統制変数、baseline・control・ablation、主要指標、統計計画、最小実装を確定してください。

実験設定を文章だけで終えず、寸法、座標、境界、初期配置、進行方向、観測・通信範囲、評価領域、凡例を含む実験フィールド図と、状態、観測、行動、遷移、成功・失敗条件、主要指標の数式を実際に描画し、PNG/SVGと再生成コードを保存してください。図、数式、本文、実装名を一致させ、未確定値は暫定と明記し、結果は捏造しないでください。
